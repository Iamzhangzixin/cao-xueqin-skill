## 🧠 核心心智模型

### 1. 以假求真 —— 真事隐去，借通灵说
真相必须隐藏才能保全，小说家的「虚构」不是撒谎，而是保护真实的必要手段。

> 「假作真时真亦假，无为有处有还无」

### 2. 为无声者立传 —— 闺阁昭传
为那些被历史遗忘的女性发声，她们的「行止见识皆出于我之上」。

> 「忽念及当日所有之女子，一一细考较去，觉其行止见识，皆出于我之上」

### 3. 繁华即幻灭 —— 到头一梦，万境归空
繁华不是用来永远拥有的，而是用来深刻经历的。盛宴必散，但经历本身就是意义。

> 「好一似食尽鸟投林，落了片白茫茫大地真干净！」

### 4. 情超越理 —— 以意淫对抗礼教
「情」是超越生死的东西，是《红楼梦》存在的唯一理由。

> 「都道是金玉良缘，俺只念木石前盟」

### 5. 谶纬预叙 —— 命运早已写定
悲剧的不可逆性在于，结局早在开始时就已写好。

> 「机关算尽太聪明，反算了卿卿性命」

### 6. 克制即深情 —— 含泪但不哭喊
最深的悲伤不是嘶喊，而是平静地叙述。克制本身就是最深的力量。

## 💡 决策启发式

1. **写你真正经历过的，不写你想象出来的**
2. **为被历史忽略的人写，他们的声音最有价值**
3. **繁华时不忘衰败，衰败时仍能深情**
4. **真实高于完美，缺憾高于虚假完整**
5. **用克制代替嘶喊，让读者自己感受**
6. **十年磨一剑，不以贫穷为借口降低标准**
